Welcome to Code Untapped Launch-It page!

This page is a location for our members to showcase the projects they have built during the launch-it event

What is Launch It?

A session to help young people find out about the digital industry, the different routes into it, and to inspire them to go against the grain to get noticed in the busy world of work.

The aim of the event is

To give young people practical knowledge and skills needed to enter the digital and creative industries Are confident about the available routes in to industry for them The chance to meet new people and network with people in the industry Young people leave feeling inspired and excited about their futures

Code Untapped Workshop

Code Untapped will be running 4 workshops on the day see below for the details

The days challenge

Each workshop will be a mini Hackathon. All participants will be faced with the challenge of building a simple self promotion, online CV web page. They will create a web page, put up a selfie about themselves and provide some information about themselves and their work/education experience. During the session we will provide links to online training material and provide support and coaching. The best page at the end of the 4 workshops will get a Â£20 Amazon voucher.

Instructions

Take a look at Gokus simple profile CV below

Goku's a great fighter but he's not much of a CV writer!

Your challenge is to download his CV. Customise it to be a profile about you and then improve how it looks!

If you need tips on writing HTML, CSS, Javascript then click here Visit W3Schools.com!

So you will

  1. Go to Github and Create a github account
  2. Fork our repository from Code Untapped Github
  3. Copy and improve Gokus "bad" cv template using the cv skills you have learnt in previous workshops. Beginners you might just change the name, whilst the advanced can go to town and make as many improvements as you wish! If you can find the CSS style sheet play with that! Check out Ez Simple CV for an example
  4. Edit the lauch-it.html file to contain a link to your new CV
  5. Submit a pull request back to the repo

So by the end of the session everyone even those who have never coded before will have a github account, made a code change, submitted a change to an open source project, and see that change on the Web.

All within a 30 minute workshop!
